CQOI2016K远点对

已知平面上$n$个点的坐标，求欧氏距离下的第$k$远点对的距离。

（$1\leq n \leq 100000,1\leq k \leq 100$）

维护一个大小为$2k$的小根堆，如果可能的最大距离小于堆顶就剪掉，其它和平面最近点对一样，所有点都问一次即可。

BZOJ3489A simple rmq problem

给定一个长度为$n$的序列。有$m$个询问，每次询问要求在$[l,r]$之间找到一个在这个区间内只出现过一次的最大的数。

（$1\leq n \leq 100000,1\leq m \leq 200000$）

每个点有四个维度：{前一个与之相同的位置，本身的位置，本身的值，后一个与之相同的位置}。

对于询问[$l,r]$，相当于查询$L=(0,l,0,r+1)$与$R=(l-1,r,inf,n+1)$之间所有元素第三维（点值）的最大值。每次递归到一个点，若其不被$L，R$严格包含，则直接返回；否则更新$L$的第三维并继续递归下去。最后$L$的第三维就是本次询问的答案。
